# AI Usage

AI tools were used in a limited capacity to assist development and troubleshooting.

## Tools Used

openAI, gitHub Copilot

## Areas Where AI Was Used

AI assistance was used primarily for:

- debugging Hugo template issues
- troubleshooting Netlify deployment configuration
- improving code readability and structure
- reviewing template logic for potential errors

## Areas Implemented Manually

The following aspects were implemented and verified manually:

- CMS schema design
- overall project architecture
- Sanity data modeling
- Hugo page templates and routing
- Node.js data export script
- UI layout and styling
- deployment configuration
- functional testing of the full system

AI was used as a development aid, but all implementation decisions and validation were performed by the developer.
